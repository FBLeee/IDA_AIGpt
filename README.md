# IDA_AIGpt
集成gpt-3.5-turbo-16k模型到IDA插件中，利用GPT来解释IDA反汇编代码

## 使用步骤 
### 1.将脚本放入插件中
将Model_GPT_16k中的gepetto-locales 和 Fbleee_16k.py脚本放到 IDA 插件文件夹 ( $IDAUSR/plugins)
